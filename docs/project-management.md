# Gestión del Proyecto

## Herramienta utilizada

Para organizar el desarrollo de Top 5 de Todo se usa **Trello**.

🔗 [Tablero en Trello](https://trello.com/b/gbZFPsx6/fullstack-project)

## Estructura del tablero

El tablero tiene 5 columnas:

- **Backlog** — funcionalidades definidas pero no empezadas
- **Todo** — tareas listas para empezar en el sprint actual
- **In Progress** — en desarrollo activo
- **Review** — terminadas, pendientes de revisión
- **Done** — completadas

## Tarjetas

Hay 7 tarjetas, una por bloque funcional:

1. Setup inicial
2. Modelo de datos
3. Estado y persistencia
4. Pantalla Rankings
5. Crear y editar ranking
6. Detalle y Favoritos
7. Estadísticas y componentes UI

## Flujo de trabajo

Cada tarjeta se mueve de columna conforme avanza el desarrollo. Las subtareas dentro de cada tarjeta se marcan como completadas antes de mover la tarjeta a Review.
